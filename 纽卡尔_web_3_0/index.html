关于我们

我们是一家专业的阿尔兹海默症诊疗平台.

  • 深度学习智能诊疗

    通过与功能MRI影像结合，将结构MRI 显示的全脑体素、皮质厚度以及海马体形状体积特征输入深度卷积网络入口，通过算法的深度学习，提取到的特征再输入分类器中，就完成了 AD/MCI的诊断

  • 分布式云计算

    我们的云平台采用分布式的架构方式，合理 利用了集群的运算优势，将计算资源按需分配给用户，使集群的运算时间大幅度减少， 保证了系统对用户影响的实时分析

  • 医学影像专利

    系统建设依靠于基于深度学习的磁共振影像特征提取及分类方法，提高磁共振成像信噪比的方法及其系统等10多项专利，为系统诊断方法的科学性和准确性提供理论支撑

  • 云pacs系统

    借助于华西医院放射科3万例AD患者影像资源作为训练模板，利用云pacs系统搭建广域影像资源共享平台，同时借助于核心技术的三分类识别优势，以及诊疗系统的随访功能，纽卡尔平台将给国内长期受阿尔兹海默症困扰的老年群体带来福音。

技术路线

纽卡尔是一个基于中国人大脑模型数据库，结合云计算和人工智能核心技术，精准评估大脑结构的影像学分析诊断平台。

预处理模块

首先是部署在医院的图像预处理设备。这一设备可以与医院的PACS 系统直接通信，访问相应的MRI图像。设备将利用自身所带的算法，将庞大的原始MRI图像进行压缩，并且完成滤波、配准和标准化的一系列 图像预处理过程。最后将处理好的较小的图像上传至本公司的中央服务 器进行诊断。

神经网络诊断模块

本公司的服务器将配备GPU或TPU进行神经网络计算，加速诊断过程。 服务器可以在数秒内将诊断结果返回给患者或医院。 得益于现代半导体工业的发展，人们逐步将深度学习的实现从软件层 面转移至更高效的硬件层面。本公司也拟在未来使用自己的机器视觉芯片，直接在医 院终端处理识别MRI图像，以实现更高效的诊断，

图像输入分类器

纽卡尔提出了一种基于深度学习的磁共振影响特征提取方法， 有效解决了传统的磁共振图像分类过程中需要人工选择特征输入致使分类精度不高的问题， 其通过多层卷积神经网络的非线性映射， 自动得到高度可分特征/特征组合用于分类，并且可以不断优化网络结构得 到更好的分类效果。

竞品分析

现有的用基因诊断的方法诊断阿兹海默症的准确性高，也能从基因层面提供较为可靠的治疗方案，但操作过程繁杂，工作量大，花费高。

用量表基于量表实现AD 诊断简单易行，但是主观性较强，患者的受教育 程度、文化背景以及医生的个人主观习惯都会影响诊断的可靠性。
